## Features

- Advanced movie search
- Genre filtering (20+ categories)
- Custom sorting (top rated, latest, etc.)
- Responsive design (mobile, tablet, desktop)
- Integrated contact system
- Advanced security (rate limiting, CORS, headers)
- Automatic deployment on Vercel

## Live Demo

Access: [https://cineworld-site.vercel.app](https://cineworld-site.vercel.app)

## Technologies

- **Front-end:** HTML5, CSS3, JavaScript Vanilla
- **Back-end:** Node.js, Express.js
- **API:** TMDB (The Movie Database)
- **Hosting:** Vercel
- **Security:** Helmet, Rate Limiting, CORS
